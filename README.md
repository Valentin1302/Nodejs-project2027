# Nodejs-project2027

## Configuration Google Calendar (local)

Ce projet utilise Google Calendar via OAuth2.

Les fichiers suivants ne sont pas versionnés :
- google-credentials.json
- token.json
- .env

### Pour tester le projet :

1. Créer un projet Google Cloud
2. Activer l'API Google Calendar
3. Créer des identifiants OAuth2
4. Télécharger `google-credentials.json`
5. Lancer le serveur une première fois pour générer `token.json`

Ces fichiers sont fournis séparément pour la démonstration.
